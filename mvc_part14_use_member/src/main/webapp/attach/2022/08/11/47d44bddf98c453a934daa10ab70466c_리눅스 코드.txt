pwd : 현재 작업 위치

ls : 현재 디렉토리에 뭐가 있는지 알려줌

ls -l -a : -l -> 자세히 // -a : 숨김파일까지 모두 보여줌
=> 결과값에 맨 앞에 d가 있으면 디렉토리 / -이 있으면 파일
--> ls -l을 줄여서 ll로 자주 사용

ls -alS  : 파일 크기 순으로 보여줌

ls - alh : 사용 용량이 얼만지 알려줌

ls -alt : 시간순으로 보여준다네

touch 이름.확장자 : 해당 이름을 가진 파일 생성

mkdir 이름 : 해당 이름으로 가진 디렉토리 생성

rmdir 이름 : 해당 이름을 가진 디렉토리 삭제

rm -r : 해당 폴더에 파일이 있더라도 삭제

cp 원본파일 경로/copy.확장자 : 해당 파일을 해당 경로에 복사

cd : 디렉토리 이동

cat : 전체 내용 출력

git remote add origin https://github.com/BSHaRu/test_git.git



head : 위에서부터 10줄만 출력

head -n 숫자 : 위에서부터 숫자만큼 출력

tail : 마지막 부분 출력

vi : 파일 편집
-> a 혹은 i를 누르면 해당 파일에 텍스트를 추가 할 수 있음
-> 편집 다끝나면 esc 누르면 다시 돌아감
-> :q! 를 입력하면 해당내용 저장안하고 편집 종료
-> :set number : 문서의 라인번호 볼 수 있음
-> :wq : 저장하고 종료해줌
[ 편집모드 아닐때 ]
-> dd :  커서 위치에서 한줄 삭제
-> yy : 한줄 복사
-> p : 복사한거 붙여넣기
